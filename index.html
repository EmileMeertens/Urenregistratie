ⓘ Hoe stel je het Apps Script in?
  1. Ga naar script.google.com → Nieuw project
  2. Plak het Apps Script (zie SETUP.md) in het editorvenster
  3. Sla op, klik Implementeren → Nieuwe implementatie
  4. Type: Web-app · Toegang: Anyone
  5. Klik op Implementeren en kopieer de URL
  6. Plak de URL hierboven in het URL-veld
  7. Kies Google Sheets als opslagmethode en sla op
